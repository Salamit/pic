I am completing this book as part of the required reading for the Harvard CS50x course.

Having searched for answers to the exercises and found very little help, I have put my answers on my github page for anyone who needs them.

You will find the book examples as well.

You will find the documents organised by chapter. Every chapter has a folder for the book examples and a separate folder for my answers to the exercises.

Feel free to message me if the code could be better.

Long live bytes!
